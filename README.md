# Work Session Tracker

## Problem
In today's fast-paced environment, users frequently switch between tasks. 
Each switch causes loss of context, leading to wasted time and reduced productivity. 
There is no simple system that helps users resume their work instantly.

## Solution
We built a Work Session Tracker that acts as a personal mission control dashboard. 
It allows users to create sessions, pause/resume tasks, and quickly restore context 
so they can continue working without losing focus.

## Features
- Create work sessions with task name and notes
- Pause and resume sessions
- View current active session
- Track all sessions in a timeline
- Visualize time spent on tasks

## Tech Stack
- Python
- Streamlit
- Pandas (optional)

## How to Run
pip install streamlit
streamlit run app.py
